jgfx — examples

A JavaScript / WebGPU port of cgfx. Each example mirrors its cgfx twin.

WebGPU is not available in this browser. Use a recent Chromium-based browser (or enable WebGPU) and serve this page over http://localhost.
Serve from the repo root, e.g. python3 -m http.server or npx serve, then open /examples/. See the documentation for guides and the API reference, and PLAN.md for the roadmap.
Single-file build: node build.mjs → dist/ (verify with dist/smoke.html).
